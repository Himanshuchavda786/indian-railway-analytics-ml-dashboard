## 📖 Project Overview

Train Journey Analytics & Duration Prediction is an end-to-end Data Science and Machine Learning application built using Python and Streamlit. The project provides comprehensive journey analytics, interactive visualizations, and predictive modeling to estimate train journey duration based on travel distance and number of stops.

The application transforms raw railway journey data into meaningful insights through advanced analytics dashboards and machine learning predictions, enabling users to understand travel patterns and forecast journey durations effectively.

---

## 🎯 Problem Statement

Accurately estimating train journey duration is essential for efficient travel planning and transportation analysis. Traditional estimation methods may not fully leverage historical journey data and operational patterns.

This project addresses the challenge by:

- Analyzing historical journey data
- Identifying key factors affecting travel duration
- Visualizing operational trends
- Predicting journey duration using Machine Learning
- Providing an interactive decision-support dashboard

## 📂 Dataset Information

The dataset contains train journey-related attributes including:

| Feature | Description |
|----------|------------|
| Total_Distance | Total travel distance covered |
| Total_Stops | Number of intermediate stops |
| Journey_Duration_Hours | Total duration of the journey |
| Route Information | Journey route details |
| Operational Data | Additional travel information |
